Comp 40 - Machine Structure and Assembly Language

Comp 40 is a C-based programming course that challenged me to deeply understand the intimate relationship between hardware and software. To achieve this understanding, I completed 9 major assignments, which ranged from high-level and abstraction heavy to as low a level as working with assembly language and manually packing bits into 64-bit integers.

Although this course was by far the most work-intensive class I've ever taken, it was also the most rewarding, as coding at such a high intensity for an entire semester really helped me turn the corner from introductory programming to programming as a professional, with a deep understanding of computer hardware and the way data and code are represented at the machine level.

  • Assignment 4: Arith
    • Implemented an algorithm that takes a PPM image and compresses the image to a new format consisting of 64-bit "words"
    • Learned about packing and unpacking multiple values (signed and unsigned)into/from words, as well as other bitwise arithmetic (hense the name of the assignment)
  • Assignment 6: UM
    • Implemented a universal machine, which consisted of an instruction set that is executed using general memory segmented into 32-bit words and 8 registers.
    • All future assignments required me to modify or build on this code in some way, including optimizing performance, writing a macro assembler, and implementing a RPN calculator utilizing only the instructions and macros defined in earlier assignments.

Comp 105 - Programming Languages

Comp 105 introduces students to the study of programming languages and their implementation. This course was my first introduction to functional programming. Topics covered included abstract syntax, type systems/polymorphism, lambda calculus, and garbage collection, as well as others.

  • Assignment 7: Garbage Collector

Comp 150 - Computational Models in Cognitive Science

Comp 150 was a survey of various computational models and modeling techniques in the context of cognitive science. Topics covered included connectionist models, bayesian models, dynamic system models, logic models, and classical cognitive architectures.

For the final project, student were required to create their own computational model of some congitive phenomenon - I chose to make a connectionist model of selective attention.

Comp 170 - Computational Theory

Comp 170 is an introduction to computation theory. Topics covered include Turing machines, recognizability/decidability, reductions, oracles, time complexity, space complexity, context-free grammars, and more.

Comp 175 - Computer Graphics

Comp 175 introduces students to the fundamentals of computer graphics. Over the course of 5 assignments, I built from scratch a recursive ray tracer that renders 3D scenes, including reflections, shadows, specular highlights, texture mapping, and anti-aliasing.
